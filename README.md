# 📅 Algenda

**인공지능 기반 자동 회의록 생성 및 지능형 시각화 관리 플랫폼**

비정형 회의 음성을 정형화된 데이터로 변환하고, LLM을 활용해 논의 구조를 시각화하여 효율적인 의사결정을 돕는 지능형 지식 관리 시스템입니다.

---

## ✨ 주요 기능

* **음성 인식 및 화자 분리**: OpenAI Whisper 모델을 활용하여 한국어 음성을 텍스트로 변환하고 발언자를 식별합니다.
* **지능형 핵심 요약**: 회의 전체 맥락을 분석하여 핵심 안건과 일정, 담당자 등의 주요 정보를 자동으로 추출합니다. 
* **논리 구조 시각화**: 분석된 데이터를 바탕으로 회의의 흐름을 마인드맵이나 순서도 형태의 시각적 자료로 생성합니다. 
* **지능형 지식 인터페이스**: RAG 기술을 적용하여 과거 회의 기록을 기반으로 사용자의 질문에 정확한 근거를 바탕으로 답변합니다.

---
