# 📅 Algenda (AI + Agenda)

**인공지능 기반 자동 회의록 생성 및 지능형 시각화 관리 플랫폼**

비정형 회의 음성을 정형화된 텍스트 데이터로 변환하고, LLM을 활용해 논의 구조를 시각화하는 지능형 회의 지식 관리 시스템입니다.

---

## 🚀 프로젝트 개요 (Project Overview)
***목표**: STT(Speech-to-Text)와 LLM 기술을 결합하여 회의 중 발생하는 정보의 휘발성 문제를 해결하고, 비정형 데이터를 정형화된 지식 자산으로 변환합니다.
* **핵심 기능**:
    * **음성 인식**: OpenAI Whisper 모델을 활용한 한국어 음성 인식 및 화자 분리.
    * **지능형 분석**: 핵심 안건 요약 및 주요 정보(날짜, 시간, 담당자 등) 추출. 
    * **논리 구조 시각화**: React Flow 기반의 마인드맵/순서도 자동 생성.
    * **RAG 기반 챗봇**: 과거 회의 이력을 기반으로 정확한 근거를 답변하는 지능형 인터페이스.
